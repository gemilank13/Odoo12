Base Account by Alphasoft for Odoo V.12

Features:

  • Menu Items: Invoicing, Cash & Bank, Taxes, Reports, Invoice Analysis
  • Wizard: Compute selected invoice to compute taxes, Unposted selected Journal Entries, Set to Draft selected Invoice
  • Account Journal Added Type (Advance Sale, Advance Purchase)
  • Account Invoice Added Fields Attention, Aged Due, Signature, Journal Bank
  • Account Move Added Product for Easy Searching at Journal Entries
  • Account Move Line Added Aged Due
